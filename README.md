# Banana Ripennes Detection

Penentuan tingkat kematangan buah pisang secara manual masih bergantung pada pengamatan manusia sehingga hasilnya dapat bersifat subjektif. Perkembangan Computer Vision memungkinkan proses identifikasi tingkat kematangan buah dilakukan secara otomatis melalui pengolahan citra digital. Pada proyek ini digunakan metode YOLOv5 untuk mendeteksi tingkat kematangan buah pisang ke dalam tiga kategori, yaitu busuk, matang, dan mentah.

- dikcy triharyadi
- bima kurnia sandi
- Lulu atun hasna
- umi arina
- umi saskia
- novita rahmawati
